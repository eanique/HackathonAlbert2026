**PROBLÈME**

Comment identifier un navire de manière fiable et passive (sans dépendre de son système AIS), en exploitant aussi sa radio signature (émissions électromagnétiques involontaires ou volontaires) ?
Dans un environnement maritime où les navires peuvent désactiver, falsifier ou manipuler leur AIS (Automatic Identification System), il est crucial de disposer d’une méthode alternative pour : 

+ Identifier un navire (nom, type, pavillon, destination).
+ Localiser sa position (même en l’absence de données AIS).
+ Détecter des anomalies (changement de nom, faux pavillon, activité suspecte).
+ Créer une base de données de signatures radio pour une identification passive.

**SOLUTION**

Concevoir une solution autonome, à partir d’outils open et sources de données OSINT, passive et automatisée, capable d’identifier un navire uniquement à partir de sa radio signature, sans dépendre de sa coopération (ex. : AIS activé).

+ Capturer la radio signature
+ Extraire des caractéristiques uniques
+ Modéliser les signatures
+ Automatiser l’identification
+ Valider la solution
